## Integrantes

## Descripción del proyecto
Este proyecto consiste en un sistema web CRUD para la administración de autos. Permite registrar, consultar, modificar y eliminar información de vehículos almacenada en una base de datos MySQL.

## Problema que resuelve
El sistema facilita la gestión de los registros de autos, evitando llevar la información de forma manual y permitiendo acceder a ella de manera rápida y organizada.

## Funcionalidades principales
- Registrar autos.
- Listar autos registrados.
- Editar información de un auto.
- Eliminar autos.
- Validación de datos mediante JavaScript.
- Conexión a una base de datos MySQL utilizando PHP.
